Hakkımda

Bilişim Sistemleri Mühendisliği 2.sınıf öğrencisiyim. Yazılım alanında kendimi sürekli geliştirmeyi, yeni teknolojiler ve programlama dilleri öğrenmeyi seviyorum. Projeler üzerinde çalışmak, hem pratik becerilerimi artırıyor hem de farklı bakış açıları kazanmama yardımcı oluyor. Öğrenmeye açık, çözüm odaklı ve meraklı bir yapıya sahibim.

Bu alandaki yolculuğumda, teorik bilgilerimi pratik projelerle pekiştirmeye ve sektördeki güncel gelişmeleri yakından takip etmeye özen gösteriyorum. Özellikle web geliştirme ve veri analizi konularına derinlemesine eğilmek istiyorum.

  • Programlama Dilleri: Python, Java, JavaScript
  • Web uygulamaları geliştirme konusunda tecrübe kazanmak.
  • Veri bilimi ve yapay zeka alanlarındaki yenilikleri takip etmek.

Yeteneklerim

Projelerimde ve öğrenme sürecimde odaklandığım temel teknoloji ve metodolojiler:

Web Geliştirme Temelleri

HTML, CSS ve temel JavaScript kullanarak kullanıcı dostu web arayüzleri oluşturma. Python ve Django framework'ü ile web uygulamalarının backend tarafında çalışma ve temel CRUD (Create, Read, Update, Delete) işlemleri geliştirme deneyimi.

Mobil Uygulama Geliştirmeye Giriş

Java programlama dili ve Android Studio geliştirme ortamını kullanarak temel seviyede Android mobil uygulamaları tasarlama ve kodlama becerisi. Activity yaşam döngüsü, layout tasarımı (XML) ve basit veri yönetimi konularında bilgi.

Temel Programlama Dilleri

Python dilinde veri yapıları ve problem çözme pratiği. C ve C++ dillerinde temel programlama bilgisi ve algoritmik düşünme becerisi. Farklı programlama paradigmalarına (nesne yönelimli, prosedürel) aşinalık.

Deneme

Konteynerleştirme ve Versiyon Kontrolü

Docker kullanarak uygulamaları konteynerleştirme ve geliştirme ortamlarını izole etme konusunda temel bilgi. Proje kodlarını yönetmek ve takım çalışmasına uyum sağlamak için Git versiyon kontrol sistemini aktif olarak kullanma.

Veritabanı Temelleri

İlişkisel veritabanı mantığını anlama (örn: SQLite, PostgreSQL). Django ORM gibi araçlar vasıtasıyla Python üzerinden temel veritabanı işlemleri gerçekleştirme ve basit SQL sorguları yazma bilgisi.

Problem Çözme ve Öğrenme İsteği

Karşılaşılan teknik sorunlara analitik yaklaşma ve çözüm araştırma becerisi. Yeni teknolojileri (örn: kütüphaneler, framework'ler) öğrenmeye ve projelerde uygulamaya açık olma. Algoritmik düşünme yeteneğini geliştirme.

Portföy

Eğitimim süresince edindiğim teorik bilgileri pratiğe dökmek ve yeni teknolojileri öğrenmek amacıyla çeşitli projeler geliştirdim. İşte öne çıkan bazı çalışmalarım:

Dinamik CV ve Portföy Web Sitesi

Kişisel bilgilerimi, eğitim geçmişimi, becerilerimi ve projelerimi sergileyebileceğim dinamik bir web sitesi. (Şu an ziyaret etmekte olduğunuz site).

Benim Rolüm:
  • Projenin fikir aşamasından tasarımına, backend geliştirmesinden Docker ile konteynerleştirilmesine kadar tüm süreçleri tek başıma yürüttüm.
  • Django framework'ü ile backend altyapısını oluşturdum.
  • HTML, CSS ile kullanıcı arayüzünü tasarladım.
  • Docker kullanarak projenin konteynerleştirilmesini sağladım.
Öğrendiklerim:
  • Docker'ı ilk kez kullanarak Dockerfile ve docker-compose yapılandırmak.
  • Bu proje ile full-stack web geliştirme, veritabanı yönetimi ve konteynerleştirme konularında pratik deneyim kazandım.

Kullanılan Teknolojiler: Python, Django, Docker, PyCharm, HTML, CSS, JavaScript, PostgreSQL

Sonuç: Kendi becerilerimi sergileyebildiğim, kolayca güncelleyebileceğim fonksiyonel bir web sitesi.

Android Mobil Eğitim Kursu Platformu

Kullanıcıların çeşitli eğitim kurslarına göz atabildiği, istedikleri kursu ve eğitmeni seçebildiği ve kurs içindeki ilerlemelerini takip edebildiği bir Android mobil uygulama prototipi.

Benim Rolüm:
  • Uygulamanın konseptini belirledim ve kullanıcı arayüzünü (UI) tasarladım.
  • Kotlin programlama dilini kullanarak uygulamanın temel fonksiyonlarını geliştirdim.
Öğrendiklerim:
  • Android Studio ortamına ve uygulama yaşam döngüsüne adapte olmak.
  • XML ile kullanıcı dostu layout'lar oluşturmak.
  • Farklı ekranlar arasında veri aktarımını yönetmek.
  • Bu proje ile mobil uygulama geliştirme temellerini, Java/Kotlin dilini ve UX prensiplerini öğrendim.

Kullanılan Teknolojiler: Android Studio, Kotlin , XML, RecyclerView

Sonuç: Temel özellikleri çalışan bir Android uygulama prototipi başarıyla geliştirildi.
